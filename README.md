如何使用:
1. 在Workflows里选中推你。
2. 点击中间的Script文件并双击进入编辑界面
3. 您应该能看到有以下文字：
#此处一定填写你的推号或者推名，例如: t2u_id = "jack" or "123456"
t2u_id = "test"
#此处一定填写你的推码，例如: t2u_transfer_password = "111111"
t2u_transfer_password = "test"

4. 按照提示修改即可，然后保存
5. 呼出Alfred输入'tu 我可以推送啦'，回车
6. 推送完成后，会有通知提醒结果
